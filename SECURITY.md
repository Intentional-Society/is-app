# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in this codebase, please report it privately rather than opening a public issue. A public issue exposes the problem to everyone before it can be fixed.

**Use GitHub's private vulnerability reporting:**
Repository → Security tab → "Report a vulnerability"

Please include:
- A description of the vulnerability and where it exists
- Steps to reproduce it
- What an attacker could do with it (impact)
- Any suggested fix, if you have one

## What to Expect

We'll acknowledge your report within a few days and aim to have a fix in place within 14 days for anything critical. We'll keep you updated as we work on it.

This is a small community app maintained by a small team — we'll be straightforward with you about timelines.

## Scope

This policy covers the application at `app.intentionalsociety.org` and the source code in this repository.

It does not cover the main website at `intentionalsociety.org`, which is a separate system.

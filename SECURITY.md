# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately via [GitHub's private vulnerability reporting](https://github.com/Intentional-Society/is-app/security/advisories/new) rather than opening a public issue. Include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof of concept
- Any suggested mitigations if you have them

## Scope

`app.intentionalsociety.org` only. The public site (`www.intentionalsociety.org`) is out of scope.

## Supported Versions

Only the deployed `main` branch is supported. There are no released versions or backports — fixes ship forward on `main`.

## Response Expectations

We aim to acknowledge reports within 48 hours and provide a resolution timeline within 7 days.
